Bem vindo ao JUnit5 com Sublime Text 4
======================================

Criei esse arquivo para estudo próprio e agora vou deixálo aqui para quem 
esta iniciando na programação e utilizando linux e sublime text. A programação
com ele fica bem mais simples quando é criado alguns arquivos de configuração.
Na pasta tem alguns arquivos que fazem essa configuração automaticamente. Veja de
acordo com sua versão java e configure da forma que deseja. Lembre-se, esse arquivo serviu para mim em um momento, não quer dizer que não possa ser alterado e melhorado.
Fique a vontade para realizar alterações da forma que achar melhor.

Não estou usando nem Maven e nem Gradle ou Kotlin, estamos usando java puro, pois
a finalidade aqui é a compreensão e dominio da ferramenta. Assim, a resposta é bem mais rápida no momento dos testes, mas nada impede de ser usado com Maven, Gradle,
Kotlin. Vai de cada programador utilizar melhor a sua ferramenta.
